# 🇨🇦 Canada Tourism Expenditure Forecasting

📌 Project Overview  
This project analyzes and forecasts international tourism expenditure in Canada using time-series models. The goal is to understand spending patterns and predict future trends.

🔍 Key Contributions  
- Analyzed tourism expenditure trends using data from Statistics Canada  
- Built and compared multiple time-series models (STL, ETS, ARIMA)  
- Evaluated model performance and selected the best-performing approach  

📊 Key Insights  
- USA is the primary source of international tourists  
- Lodging, dining, and transportation are the largest spending categories  
- ARIMA achieved the highest forecasting accuracy among tested models  

🛠️ Tech Stack  
- R (fpp3, tidyverse, ggplot2)  
- Modeling Techniques: STL Decomposition, ETS, ARIMA  
- Data Source: [Statistics Canada (Table 24-10-0047-01)](https://www150.statcan.gc.ca/n1/en/catalogue/24-10-0047-01)
